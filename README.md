# adventofcode-2016
Advent of Code 2016

## Run puzzles

You can use `run` to run the all the puzzles or a particular puzzle:

        $ ./run -h
        Usage: run [day#]

        Runs all the puzzles or optionally the puzzle for the given day

        Examples:
          ./run
          ./run 4

Solutions

* day01 - Ruby 2.3.3
* day02 - Posix sh
* day03 - Scala (tested with 2.12.0)
